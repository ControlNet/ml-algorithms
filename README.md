# Machine Learning Algorithms

R implementation of some traditional machine learning algorithms for data analysis.

Including:

- Classification
    - Bayesian Classifier
    - KNN Classifier
    - Logistic Regression
    - 3-Layer Neural Network
    - Perceptron
- Clustering
    - Gaussian Mixture Model
    - K-Means
- Regression
    - KNN Regression
    - Linear Regression
- Others
    - Bootstrap
    - Cross Validation
